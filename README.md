### 👋 About me

I'm a security researcher focused on **web application security** and
**source-code auditing** of open-source software. I find vulnerabilities,
report them through **coordinated/responsible disclosure**, and help
maintainers ship fixes.

- 🔭 Auditing open-source web apps (PHP / Python / JavaScript)
- 🛡️ Vulnerability classes: XSS, Path Traversal, SSRF, SSTI, Auth bypass, Injection
- 📨 I always report privately to maintainers first and follow disclosure timelines
